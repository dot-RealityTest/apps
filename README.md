## About

Built by [Kika](https://akakika.com) — former touring DJ, now building macOS apps with AI assistance.

All apps are:
- **Free** — no cost, no tracking
- **Native** — Swift, SwiftUI, Tauri, or Electron
- **Notarized** — safe to install on macOS

## Development

Each app has its own folder with source code, build scripts, and documentation.

```bash
# RESQ (Vite + React)
cd RESQ && npm install && npm run dev

# Focus (Tauri)
cd focus && npm install && npm run tauri dev

# FolderWardrobe (Swift)
cd folderwardrobe && swift build && swift run

# mochi (Electron)
cd mochi && npm install && npm start
```
