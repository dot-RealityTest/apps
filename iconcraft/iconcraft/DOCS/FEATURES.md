# IconCraft Features

## Core workflow

- Upload one source image in PNG, JPG, or SVG format
- Preview the icon across the required macOS output sizes
- Crop the artwork to refine framing
- Adjust padding for better balance in the icon canvas
- Upscale smaller images before export when needed

## Export formats

- `AppIcon.iconset` ZIP
- `.icns`
- `AppIcon.xcassets` ZIP for Xcode
- React Native iOS icon ZIP

## Preview and review

- Compact preview matrix for all generated sizes
- Individual PNG download from each preview tile
- Primary output focus for the 1024×1024 icon
- Workflow checklist that tracks upload, crop, upscale, and export progress

## Desktop app

- Native macOS app bundle
- DMG installer for sharing
- Local workflow with no account or cloud setup required
